# Employee CRUD Odoo v14

Aplikasi sederhana untuk mengelola data karyawan menggunakan Odoo v14.
![List Employee](image/list.png)

## Fitur
- Create data karyawan
![Create Employee](image/create.png)

- Read data karyawan
![Read Employee](image/read.png)

- Update data karyawan
![Update Employee](image/update.png)

- Delete data karyawan
![Delete Employee](image/delete.png)


## Cara Install
1. Copy folder modul ke folder addons Odoo, misalnya:
    - addons/
    - custom_addons/
2. Restart Odoo
3. Update Apps List
4. Install modul "Employee CRUD"
